Constellation Module Suites Explained

Constellation uses a pluggable architecture and is build using different module suites. A collection of different module suites form different versions of Constellation.

If you are interested to know how these module suites are built to create the different versions then see the technical documentation here.


Constellation "Core"

This is the base module which all versions of Constellation are based on. This module is the domain agnostic framework layer which includes views like the 3D Graph view, Histogram, Analytic View, Map View, Scatter Plot, Import from File etc...

Note that this version is designed to work offline.

The source code for this module is available here.

Constellation Adaptors

Designed to be built on top of "Core", provides Constellation with access to online services like various map services (such as Google Maps, Bing Maps, ArcGis Maps).

There are also plugins to query and enrich your graph using the Data Access View plugins that support GML, GraphML and Pajek file formats.

The source code for this module is available here.

Constellation Cyber Plugins

This module is developed by the Australian Cyber Security Centre (ACSC) and includes additional plugins to connect to Virus Total and MaxMind.

To use these plugins you will need to add your API key by going to Setup → Options → Constellation → ACSC.

The source code for this module is available here.

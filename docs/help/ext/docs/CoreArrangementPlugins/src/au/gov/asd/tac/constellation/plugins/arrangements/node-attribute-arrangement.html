Node Attribute

Constellation Action Keyboard Shortcut User Action Menu Icon
Run Node Attribute Arrangement Arrange -> Node Attribute

The node attribute arrangement arranges nodes based on the value of a specified node attribute. The way the nodes are arranged is dependent on whether you specify arranging in 3D or not. If you opt to arrange in 3D, nodes are layered across the z-axis with nodes that have the same specified attribute value being positioned at the same point on the z-axis. If you opt not to, nodes with the same specified attribute value are grouped together in a square grid arrangement.

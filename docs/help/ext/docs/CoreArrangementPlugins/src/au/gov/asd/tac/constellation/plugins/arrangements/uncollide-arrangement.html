Uncollide

Constellation Action User Action Menu Icon
Run Uncollide Arrange -> Uncollide
Run Uncollide 3D Arrange -> Uncollide 3D

Selected nodes are postioned such that they don't "collide" with one another. Nodes are considered "colliding" if they overlap with one another. If no nodes are selected, then all nodes on the graph will be affected.

Using uncollide will expand the graph until none of the selected nodes are colliding. If none of the nodes are colliding the begin with, nothing will happen.

Uncollide will only calculate with respect to the X and Y axis, whereas Uncollide 3D accounts for all three axis.

A composite node is a node that contains 2 or more nodes. It can be expanded to reveal the nodes contained within.

Composite Selected Nodes

To create a composite node by selection, first select all the nodes you wish to be composited, then either click Tools -> Composite Selected Nodes or right-click and select "Composite Selected Nodes".




Composite Correlated Nodes

To create a composite node by correlation, simply click Tools -> Composite Correlation Nodes. Two nodes are considered correlated if at least one transaction connecting them is of type "Correlation". Multiple nodes chained by correlation are composited together.



Expand Composite

A composite node can be "expanded", revealing the nodes contained within. This can be done by right-clicking and choosing "Expand Composite".



Contract Composite

Right-clicking any one of the expanded nodes and choosing "Contract Composite" will contract the expanded nodes back into the single composite node.



Destroy Composite

A composite node can be "destroyed", returning the contained nodes to the graph. This can be done by right-clicking and choosing "Destroy Composite".



Destroy All Composites

A tool to "destroy" all composites on the graph.
